# ResusCoach CPR

医療従事者および一般市民向けの、フィードバック機能付き心肺蘇生法トレーニングアプリのスマホ向けプロトタイプです。

## 現在の主な機能

- 一般市民 / 医療従事者のユーザー種別切り替え
- 成人BLS、院内急変、小児BLS、AED遅延などのシナリオ選択
- AHA / ERC / ILCOR / 施設プロトコルの参照プロファイル選択
- 圧迫テンポ、圧迫深度、リコイルのリアルタイム評価UI
- インストラクター向けレポート画面
- Plus / Delta形式のディブリーフィング支援
- PDFレポート出力を想定した導線

## 開き方

`index.html` をブラウザで開くと確認できます。

ローカルサーバーで確認する場合は、Node.jsが利用できる環境で次を実行します。

```bash
node dev-server.mjs
```

その後、`http://127.0.0.1:4173` を開きます。

## 注意

このプロトタイプは教育・訓練支援を目的とした画面設計案です。実運用では、最新の国際ガイドライン、地域の法令、所属施設のプロトコル、認定コース要件、医療機器連携要件に基づくレビューが必要です。
